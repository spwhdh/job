# 📌 채용공고 제작 도구 사용 설명서

## 🚀 시작하기

### 1단계: 프로그램 실행
1. 파일 탐색기에서 `D:\project\job` 폴더를 엽니다
2. **`editor.html`** 파일을 더블클릭합니다
3. 브라우저(크롬, 엣지 등)에서 채용공고 제작 도구가 열립니다

---

## 💡 화면 구성

### 왼쪽: 입력 영역
- **이런분들에게 추천합니다**: 채용 대상을 설명하는 내용
- **담당 업무**: 직무별 주요 업무 내용
- **자격 요건**: 필수 자격 조건
- **우대 사항**: 추가 우대 조건
- **근무 조건 및 연봉**: 고용 형태, 연봉, 근무지, 복리후생 등

### 오른쪽: 실시간 미리보기
- 상단 고정 이미지 (회사 소개 등)
- 입력한 내용이 표시되는 영역
- 하단 고정 이미지 (오피스 사진, 복리후생 등)

## ✍️ 작성 방법

### 기본 작성 규칙
1. **줄바꿈으로 항목 구분**: 각 항목을 **엔터(Enter)**로 구분하여 입력하세요
2. **실시간 반영**: 입력하는 즉시 오른쪽 미리보기에 반영됩니다
3. **자동 저장**: 5초마다 자동으로 저장되므로 안심하고 작업하세요

## 🎨 주요 기능

### 1. 실시간 미리보기
- 왼쪽에서 입력하면 **즉시** 오른쪽에 반영됩니다
- 오타나 실수를 바로 확인할 수 있습니다

### 2. 자동 저장
- **5초마다 자동으로 저장**되므로 작업 내용이 사라지지 않습니다
- 브라우저를 닫았다가 다시 열어도 내용이 유지됩니다
- 수동 저장: **Ctrl + S** 키를 누르면 즉시 저장됩니다

### 3. 확대/축소 (줌)
- 오른쪽 상단의 **🔍-** 버튼: 축소
- 오른쪽 상단의 **🔍+** 버튼: 확대
- 미리보기 크기를 조절하여 편하게 작업하세요

### 4. 이미지 다운로드
- 우측 상단의 **📥 이미지 다운로드** 버튼 클릭
- 완성된 채용공고가 PNG 이미지로 저장됩니다
- 파일명: `채용공고_년월일_시분.png`
- 단축키: **Ctrl + D**

## 💾 데이터 관리

### 자동 저장
- 브라우저의 로컬 스토리지에 자동으로 저장됩니다
- 컴퓨터를 껐다 켜도 데이터가 유지됩니다
- 단, **브라우저 데이터를 삭제**하면 함께 삭제되니 주의하세요

### 데이터 백업 방법
1. 모든 내용을 완성한 후 **이미지로 다운로드**
2. 또는 왼쪽 입력 내용을 복사하여 별도 파일에 보관

### 새로운 공고 작성
- 기존 내용을 지우고 새로 작성하려면:
  1. 각 입력창의 내용을 선택 후 **Delete** 키
  2. 또는 브라우저 개발자 도구에서 로컬 스토리지 삭제

## ⚠️ 주의사항

### 1. 이미지 파일 필수
- `jop top.png` (상단 이미지)
- `jop under.png` (하단 이미지)
- 이 두 파일이 반드시 같은 폴더에 있어야 합니다

### 2. 인터넷 연결 필요
- 이미지 다운로드 기능은 외부 라이브러리(html2canvas)를 사용합니다
- 첫 다운로드 시 인터넷 연결이 필요합니다
- 이후에는 오프라인에서도 작동합니다

### 3. 브라우저 권장
- **크롬(Chrome)** 또는 **엣지(Edge)** 브라우저 사용 권장
- 인터넷 익스플로러는 지원되지 않습니다

### 4. 데이터 보안
- 모든 데이터는 컴퓨터 내부(브라우저)에만 저장됩니다
- 외부 서버로 전송되지 않습니다

## 🔧 문제 해결

### 미리보기에 내용이 안 보여요
→ 왼쪽 입력창에 내용이 제대로 입력되었는지 확인하세요
→ 브라우저를 새로고침(F5)해보세요

### 이미지가 깨져 보여요
→ `jop top.png`와 `jop under.png` 파일이 같은 폴더에 있는지 확인하세요
→ 파일명이 정확한지 확인하세요 (공백 포함)

### 다운로드가 안 돼요
→ 인터넷에 연결되어 있는지 확인하세요
→ 브라우저의 팝업 차단을 해제하세요
→ 다운로드 폴더 권한을 확인하세요

### 저장된 내용이 사라졌어요
→ 브라우저 쿠키/캐시를 삭제했는지 확인하세요
→ 다른 브라우저로 열었는지 확인하세요 (크롬에서 작성한 내용은 엣지에서 안 보임)
→ 앞으로는 주기적으로 이미지로 다운로드하여 백업하세요
